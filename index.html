Comment configurer ?

Configurer Firebase

  1. Allez sur console.firebase.google.com
  2. Créez un nouveau projet
  3. Dans le menu gauche → Authentication → Sign-in method → activez E-mail/Mot de passe
  4. Allez dans Paramètres du projet (⚙) → Vos applications → ajoutez une app Web </>
  5. Copiez les valeurs firebaseConfig dans le fichier firebase-config.js
  6. Dans Firebase Console → Authentication → Paramètres → Domaines autorisés → ajoutez votre domaine GitHub Pages (votrenom.github.io)

Sans configuration Firebase, l'application fonctionne en mode invité (sans compte).
